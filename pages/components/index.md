# Components

This section contains all the components available in the **CDS** component library.
Each component is documented with 3 sections:

- **Preview**: A visual representation of the component. The preview is interactive and allows you to change the component's properties. You can also view the source code of the component easily.
- **Attributes**: A list of all the attributes that can be used to customize the component. This includes property name, description, attribute name, type and default value
- **Methods**: A list of all the methods that can be used to interact with the component. This includes method name, description and the return type.
