
# Slider

## Attributes
Property | Description | Attribute | Type | Default
--- | --- | --- | --- | ---
value | The value of the slider The value is a number between 0 and 100 | value | `number{:ts}` | `0{:ts}`
disabled | Disables the slider | disabled | `boolean{:ts}` | `false{:ts}`
step | The step of the slider | step | `number{:ts}` | `1{:ts}`
min | The minimum value of the slider | min | `number{:ts}` | `0{:ts}`
max | The maximum value of the slider | max | `number{:ts}` | `100{:ts}`
