## Attributes
Property | Description | Attribute | Type | Default
--- | --- | --- | --- | ---
label | The label of the input field If no label is provided, the placeholder will be used as label | label | `string{:ts}` | __required__
type | The type of the input field | type | `string{:ts}` | `"text"{:ts}`
placeholder | The placeholder of the input field | placeholder | `string{:ts}` | __required__
value | The value of the input field | value | `string{:ts}` | `""{:ts}`
disabled | Disables the input field | disabled | `boolean{:ts}` | `false{:ts}`
